# Gaming Rebuilds

A meta-repository for game rebuilds using agentic frameworks.

## What Is This?

This project explores rebuilding classic games using modern AI agent architectures.
Each rebuild is an independent repository linked here as a git submodule.

The shared [Agent Knowledge Base](AGENTS.md) captures best practices and design
patterns used across all rebuilds.

|---|---|---|

## Goals

- Experiment with multi-agent architectures applied to game logic
- Build a reusable library of agent patterns for turn-based and strategy games
- Document lessons learned for the broader agentic AI community
