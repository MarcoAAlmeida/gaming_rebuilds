# Karriers

**Repository:** [MarcoAAlmeida/karriers](https://github.com/MarcoAAlmeida/karriers)

**Submodule path:** `rebuilds/karriers/`

## Overview

Karriers is a carrier-based strategy game rebuild using agentic frameworks.
The goal is to model game logic, turn management, and AI opponents as a
multi-agent system.
